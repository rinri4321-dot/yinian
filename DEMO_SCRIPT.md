# 壹念 — 演示视频脚本（3 分钟）

## 准备工作

1. OBS Studio 录屏软件 — https://obsproject.com/
2. 启动项目：`npm run dev`
3. 浏览器打开 http://localhost:3000
4. 清理浏览器数据（或隐私窗口）— 确保引导页会弹出
5. 深色模式关闭（浅色模式录屏效果更好）

---

## 时间线

### 0:00 — 0:20 开场：问题

> 画面：首页空状态
> 旁白：信息过载、选择太多、不知道每天该干什么——这是我们这代人的问题。焦虑和拖延是一个循环。我查了大量临床心理学研究，发现有一套经过 RCT 验证的方法可以打破它。所以我用 AI 做了壹念。

### 0:20 — 0:40 引导页

> 点击进入 → 滚动引导页 3 屏
> 旁白：壹念不管理你的任务。它每天帮你清空大脑，找到唯一重要的事，然后让你容易开始。背后是 7 项临床研究，从认知卸载到执行意图。

### 0:40 — 1:30 核心流程（快速走完）

> 点"开始清空大脑" → 进入 /today
> 
> 步骤 1 — 清空：随意输入几行"要做PPT但没头绪\n担心下周汇报\n该给妈妈打电话\n想学编程不知从哪开始"
> 
> 点"清空完毕，帮我看看" → AI 分析动画 → 步骤 2
> 
> 旁白：AI 自动把你的思绪分类——担忧、待办、想法、提醒。然后追问一个帮你聚焦的问题。
> 
> 步骤 3 — 聚焦：选一件事，写 "做一个计划大纲"
> 
> 旁白：如果今天只能完成一件事，哪件事让其他事都变得不重要？
> 
> 步骤 4 — 定意图：填时间"上午9点"、地点"书房桌前"
> 
> 旁白：研究表明把目标写成"如果到了 X 时间在 Y 地点就做 Z"，成功率高出 2 到 3 倍。
> 
> 步骤 5 — 执行：选 5 分钟 → 开始专注
> 
> 旁白：番茄钟计时。计时结束可以选择完成了、休息继续、或部分完成存为明天计划。

### 1:30 — 2:00 完成 + 庆祝

> 计时结束 → 点"完成了" → 跳转首页 → 庆祝动画
> 
> 旁白：完成后回到首页庆祝。然后可以做晚间回顾——记录今天达成了什么、明天最重要的事。这个闭环是壹念和普通待办清单最大的区别。

### 2:00 — 2:30 洞察页

> 点底部导航"洞察"
> 
> 旁白：壹念不只是每日练习，它帮你看到自己的模式。壹指数趋势图、每周专注柱状图、AI 生成的每周洞察。数据驱动，不是玄学。

### 2:30 — 2:50 方法论 + 结尾

> 点"壹念背后的科学依据"
> 
> 旁白：7 项经过临床验证的技术，每一项都有研究支持。这不是随便想出来的功能，是真正有效的方法。
> 
> 切回首页底部
> 
> 旁白：由一个人和 Claude Code 构建，DeepSeek 驱动 AI。它先帮了我，希望也能帮到你。

## 录屏设置

- OBS：来源 → 显示器采集（全屏）或窗口采集（浏览器）
- 分辨率：1920×1080
- 帧率：30fps
- 输出格式：MP4

## 快捷操作提示

- 录屏时把浏览器缩放调到 125% 让字更清楚
- 提前开好各页面，减少加载等待
- 说错不用停，可以后期剪（或直接一气呵成）
- 用系统自带"录音机"或 OBS 同时录麦克风
